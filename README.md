This repository is the result of a classic "nothing to do on a Sunday" itch, fueled by a lot of caffeine and a deep session with Google Gemini.
It's not a corporate enterprise solution. It's a "let’s see how deep the rabbit hole goes" kind of project. 

- Single Core Only: Keeping it lean. One core, one mission. No multicore chaos (yet).
- What you can do: Task creation, priority management, and reliable context switching are all good to go.
- Built-in Safety: Basic guards against stack overflows and race conditions are implemented. It won’t blow up on your first run (I hope).
